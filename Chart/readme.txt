
GraphPlot.ppt summarise all the graphs that explained in the meeting on 22 Mar 2021. For the plot with source code, please find the directory of the corresponding folders in GraphPlot.pdf.

1. Suffix of files
Sample images: '.png', '.pdf', '.eps, '.jpg'
Data files:'.txt', '.xlsx', '.dat'
script files: '.py', '.r', '.gnu','.plt','.p'

2. Plotting with gunplot
Please install the tool from http://www.gnuplot.info/

3. Plotting with Python
Install matplotlib from https://matplotlib.org/
Install pyplot to run the plot.


4. Plotting with R
Install R environment from https://www.r-project.org/.
To run the scripts, install the ggplot2 and dplyr packages first. Use command `rscript` + filename to run the scripts.
'RStudio'(https://rstudio.com/) is a good tool in R.
The website (https://r-graphics.org/index.html) is a good resource to lean how to draw figures in R.

5. For the plotting that cannot successfully run, please go into the script and change the corresponding dir of the data files to your local machine.
